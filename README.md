# Module: MMM-WeeklyReminder

`MMM-WeeklyReminder` is a module for [MagicMirror²](https://github.com/MagicMirrorOrg/MagicMirror). It displays recurring weekly reminders based on day and time schedules. Perfect for trash day, street cleaning, lawn care, and any other weekly recurring tasks.

**⚠️ Note:** This module is currently being refactored from MMM-CustomText. Full documentation will be available soon.

## Installation

```bash
cd ~/MagicMirror/modules
git clone https://github.com/dathbe/MMM-WeeklyReminder
```
